    Hook signature: `export function useHighlighting(editor: Monaco.editor.IStandaloneCodeEditor | null, hapStream: HapStream | null): { clearAll: () => void }`

    Internal state (refs):
    - `timeoutIdsRef = useRef<number[]>([])` — flat array of all pending timeout IDs for bulk cancellation
    - `hapCollectionsRef = useRef<Map<string, Monaco.editor.IEditorDecorationsCollection>>(new Map())` — per-hap decoration collections keyed by unique hap key (use `${Date.now()}-${Math.random()}` or an incrementing counter for uniqueness)

    - Call `model.getPositionAt(start)` and `model.getPositionAt(end)`
    - Return `{ startLineNumber, startColumn, endLineNumber, endColumn }`

    Helper function `getDecorationClassName(color: string | null)`:
    - If `color` is null, return `'strudel-active-hap'`
    - If `color` is non-null, generate a safe class name `strudel-active-hap strudel-active-hap--c${hashCode}` where hashCode is a simple string hash of the color
    - Inject a `<style>` rule for the color class if not already cached (use module-level `Map<string, boolean>`)
    - Parse the color to RGB using a temporary canvas `ctx.fillStyle` trick or by checking common formats (#hex, rgb(), named colors)
    - The injected rule: `.strudel-active-hap--c${hash} { background: rgba(R, G, B, 0.3); outline: 1px solid rgba(R, G, B, 0.5); box-shadow: 0 0 8px rgba(R, G, B, 0.3); }`

    useEffect body (deps: [editor, hapStream]):
    - Guard: if (!editor || !hapStream) return
    - Create an incrementing counter ref for unique hap keys
    - Define handler function:
      ```
      (event: HapEvent) => {
        if (!event.loc || event.loc.length === 0) return;
        const model = editor.getModel();
        if (!model) return;

        const hapKey = `hap-${counter++}`;
        const showDelay = Math.max(0, event.scheduledAheadMs);
        const clearDelay = showDelay + event.audioDuration * 1000;
        const className = getDecorationClassName(event.color);

        const showId = window.setTimeout(() => {
          const decorations = event.loc!.map(({ start, end }) => ({
            range: locToRange(model, start, end),
            options: { className, stickiness: 1 as const },
          }));
          const collection = editor.createDecorationsCollection(decorations);
          hapCollectionsRef.current.set(hapKey, collection);
        }, showDelay);

        const clearId = window.setTimeout(() => {
          hapCollectionsRef.current.get(hapKey)?.clear();
          hapCollectionsRef.current.delete(hapKey);
        }, clearDelay);

        timeoutIdsRef.current.push(showId, clearId);
      }
      ```
    - Subscribe: `hapStream.on(handler)`
    - Cleanup return:
      ```
      return () => {
        hapStream.off(handler);
        timeoutIdsRef.current.forEach(id => clearTimeout(id));
        timeoutIdsRef.current = [];
        for (const col of hapCollectionsRef.current.values()) col.clear();
        hapCollectionsRef.current.clear();
      }
      ```

    clearAll function (returned from hook):
    - Cancels all timeouts and clears all decoration collections (same logic as cleanup)
    - Stored in a ref so it can be called imperatively from StrudelEditor on evaluate()
